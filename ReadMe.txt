Thông tin project
Ngôn ngữ : Java 
Framwork: jsp servlet 
Ứng dụng : Chương trình quản lý user đơn giản 
1-tầng Controller 
- Các controller để xử lý các request và response 
- Thực hiện điều hướng view và hiển thị 
2-tầng Model
- Xử lý logic của chương trình 
- Xử lý CRUD , tương tác với DB 
3-tầng View 
- Hiển thị thông tin user 
- Cho phép search thông tin user, add user, edit user, delete user
